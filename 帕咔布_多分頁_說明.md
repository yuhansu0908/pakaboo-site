# 帕咔布官網 — 多分頁版（說明）

> 建立：2026-06-20　｜　給：Cristine

把原本的「單檔分層全屏」帕咔布官網，改成跟 Emotion Design 一樣的**多分頁架構**：乾淨封面首頁 + 各分頁，共用一份 CSS/JS 與 assets。**金色星光保留**。

## 一、檔案結構

```
Pakaboo Website/
├─ index.html      首頁（乾淨封面：標語 + 三大課程 teaser + 黃金三角 + 預約 CTA）
├─ about.html      關於（品牌故事 / 我們相信 / 我想對您說 / 分離焦慮 / 給家長的信）
├─ courses.html    課程（感官 c1 / 音樂 c2 / 專業 OT c3 / 黃金三角）
├─ contact.html    聯絡我們（地址・電話・LINE・信箱 + Google 地圖 + 社群）
├─ life.html       園區生活（一日作息 / 安心守護 / 環境實景 / 菜單）
├─ assets/
│   ├─ site.css    共用樣式（暖棕配色、導覽、星光、各區塊）
│   ├─ site.js     共用程式（導覽、漢堡選單、reveal、金色星光、「我想對您說」輪播）
│   ├─ logo.png    LOGO（首頁／頁尾／品牌故事共用）
│   ├─ course_sensory.jpg / course_music.jpg / course_ot.jpg　三堂課照片
│   └─ menu.jpg    菜單參考表
└─ _build_pages.py 產生 5 個 html 的腳本（改完內容可重跑；也可直接改 html）
```

## 二、導覽列（每頁一致）
LOGO ｜ 關於 ｜ 課程▾（感官/音樂/專業/黃金三角）｜ 園區生活▾（作息/實景/菜單）｜ 聯絡我們 ｜ **預約參觀**。手機版為漢堡全屏選單。

## 三、和原版的差異
- 原版：1 個巨大 `index.html`（claude-cowork 根目錄），滑動分層、圖片內嵌 base64。**仍在、未更動**，線上 001 也還是它。
- 新版：5 個輕量分頁，圖片抽成 `assets/` 實體檔，套用 Emotion 那套多分頁手法。兩版並存，由你決定要不要換成正式版。

## 四、內容沿用
全部文字、課程、作息、菜單、地址（福雅路 128 巷 19 號）、電話、LINE @284vivvj、IG/FB 連結，皆沿用原帕咔布內容。環境實景仍是 8 格照片版位，之後有實拍照可替換 `gallery` 區。
